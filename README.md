# Simple Weather Prediction App

This is a Shiny application that predicts weather conditions based on temperature, humidity, and cloud cover.

## Live Application

You can access the working app here:

[[https://yourname.shinyapps.io/weather-app/](https://b12qzi-marta-santos.shinyapps.io/programmingassignment14/)](https://b12qzi-marta-santos.shinyapps.io/programmingassignment14/)

## Files in this repository

- ui.R
- server.R

## How it works

Users adjust sliders for weather conditions, and the app predicts the weather instantly using predefined rules.
